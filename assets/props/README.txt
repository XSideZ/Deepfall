DROP YOUR PROP ASSETS HERE
==========================

Any .glb, .gltf, .obj, .tscn, or mesh .res file you put in this folder is
auto-added to the level editor's palette the next time you run the game.

- Let Godot finish importing new files (it does this automatically when the
  editor window regains focus) before pressing Play.
- Each file shows up in the left-hand palette by its filename.
- You can also pull in a .glb from anywhere on disk at runtime with the
  "Import .glb" button in the top toolbar (no folder copy needed).

The editor already ships with placeholder shapes (Rock, Boulder, Tree, Bush,
Crystal, Pillar) so you can start blocking out a world immediately.
