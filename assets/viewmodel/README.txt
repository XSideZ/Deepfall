FIRST-PERSON HAND VIEWMODEL
===========================

Drop ONE .glb here (e.g. root_hand.glb) and it becomes the first-person "hand"
you see in Test mode (T), replacing the placeholder sapling.

- It's auto-scaled to ~0.45 m, so the model's own scale doesn't matter.
- If there are multiple .glb files, the first one found is used.
- Let Godot finish importing the file (it does this when the editor regains
  focus) before pressing Play.
- Tell me and I'll fine-tune its position / rotation / scale in the corner of
  the view once you've got one in.

Good for this: a gnarled tree root, an alien vine, a fungal tendril, a
sprouting sapling — anything that fits "you are an alien seed growing life."
